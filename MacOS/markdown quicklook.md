# 마크다운(markdown) 문서 미리보기(quicklook)
'qlmarkdown' 프로그램을  homebrew를 사용해 설치하면 된다.
이때, GUI 환경에서 작동하는 프로그램을 brew를 사용해 설치할 때는  'cask' 라는 확장 프로그램이 필요하다.

## brew 설치
이미 설치되어있는 것으로...

## cask 설치
```bash
$ brew install cask
```

## qlmarkdown 설치
```bash
$ brew cask install qlmarkdown
```
